Git 工作流

工具安装​

git-flow-avh​

使用 brew 安装 git-flow-avh，他是 git-flow 的 avh 版本。

就可以使用 git flow 命令了

zsh plugin git-flow​

配置 ~/.zshrc 添加一个 git-flow 插件

使用​

初始化​

git flow init

gfli

开发过程命令​

  1. 建立 feature 分支
git flow feature start [branch-name]

gflfs
  1. 完成 feature 分支，并且拉出 release
git flow feature finish

gflff
git flow release start [branch-name]

gflrs
  1. 完成 release 分支
git flow release finish

gflrf

hotfix 过程命令​

  1. 建立 hotfix 分支
git flow hotfix start

gflhs
  1. 完成 hotfix
git flow hotfix finish

gflhf

查看信息​

列出存在的 feature 分支

git flow feature

gflf

列出存在的 hotfix 分支

git flow hotfix

gflh

列出存在的 release 分支

git flow release

gflr
