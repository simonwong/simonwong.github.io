Object.is 和 === 有什么区别（❌）#

-0 === +0 // trueNaN === NaN // false
Object.is(-0, +0) // falseObject.is(NaN, NaN) // true
Object.is(0, -0) // falseObject.is(0, +0) // true

类中的静态成员和动态成员区别#

  1. 静态方法：在构造函数本身上定义的方法，只能通过构造函数本身调用，new出来的对象不能够调用。
  2. 动态方法：也叫做实例方法，它是通过prototype原型对象添加的，所有的实例对象都能够继承调用
