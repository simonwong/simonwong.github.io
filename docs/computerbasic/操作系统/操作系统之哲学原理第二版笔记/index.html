什么是操作系统​

操作系统是介于硬件和应用软件之间的系统软件。

站在人类的角度看，操作系统是管理者（管理硬件资源）和魔术家（为上层应用软件提供各种魔幻般的服务）

操作系统主要功能：进程管理、内存管理、磁盘管理、IO 管理
